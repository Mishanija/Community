Для создания классов, которые будут работать с данными,
используется следующие инструменты:

1 QueryService 
- Обеспечивает соединение с базой данных
- Освобождает ресурсы, когда они больше не нужны

2 DataEntitiesService -- шаблон класса для работы с сущностью
- Знает о таблице, в которой хранится сущность, с которой
он работает

3 CreateService -- работает только со "своей" сущностью
- умеет добавлять сущности в таблицу

4 FindService -- работает только со "своей" сущностью
- должен уметь искать сущность по id

5 UpdateService -- работает только со "своей" сущностью
- обновляет сущности (как в своем контексте, так и нет)

6 DeleteService -- работает только со "своей" сущностью
- умеет удалять сущности из таблицы
