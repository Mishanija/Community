Для того, чтобы добавить в приложение дополнительную 
функциональность по загрузке файлов на сервер, используется 
следующий шаблон:

1) Новый класс реализовывает типизированный класс UploadFileService.
Для того, чтобы реализовать данных класс, нужно реализвать два метода:
1 - GetPrivateName, который должен возвращать личное имя нового файла по 
его оригинальному имени.
Примечания: Для этого можно задействовать Counter, или другой 
генератор личных имен.
2 - CopyPattern, который должен возвращать объект уже определенными
общими необходимыми параметрами (например, id пользователя, который
загружает эти файлы, id альбома, куда будут помещены файлы и т.д.)

Интрументы:

1) Counter - счетчик. Используется для подсчета порядкового номера.
Хранит значение в отдельном файле на сервере. При создании счетчика 
нужно указать файл, в котором будет храниться значение.
Можно так же указать начальное значение нового счетчика, если нужно.
По умолчанию значение равно 1000.

2) ParamReader - класс для считывания параметров (предполагается строчка
с параметром) в указанном файле на сервере. 

3) UploadFileService - класс для загрузки файлов на сервер. В качестве 
параметра конструктора нужно передать 
1 - путь директории (относительно пути сервера), куда должны быть загружены файлы.
