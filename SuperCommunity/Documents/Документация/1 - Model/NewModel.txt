Создание новой модели для строго типизированного представления
происходит по следующему шаблону:

1) Определение необходимых данных на представлении

2) Формирование новой модели на основе этих данных.
Все модели должны быть отмечены интерфейсом IModel,
который означает, что данный класс является моделью.

3) Создание класса-фабрики для построения этих моделей
ModelFactory - фабрика моделей. Строит модель IModel. Построение 
модели с использованием необходимых параметров, которые должны
быть переданы через конструктор или с использованием инициализатора
новых объектов.

4) Вызов метода фабрики BuildModel() для построения нужной модели
в контроллере приложения.

В приложении существуют разновидности моделей:
- IPageModel
Модель для строго типизированных представлений

- IReadModel 
Модель для чтения, отображения информации пользователю

- IEditModel
Модель для отображения и возможностью перехода к редактированию модели

Модели IReadModel и IEditModel относятся к моделям для работы с данными (отображение, редактирование),
которая называется IDataModel
